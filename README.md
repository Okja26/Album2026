## 📖 Overview
This interactive web application is a specialized tool for collectors to manage their 2026 sticker album. It replaces traditional paper tracking with a robust digital system that synchronizes in real-time.

The project focuses on solving the complexity of manual inventory management and optimizing the "trading" process between users through data comparison algorithms.

## 🚀 Key Features
* **Live Inventory Management:** Real-time tracking of collected, missing, and duplicate stickers using a color-coded UI.
* **Smart Trading Logic:**
    * **Visual Exchange Record:** A dual-column interface to log multiple trades simultaneously.
    * **Peer-to-Peer Comparison:** Automated matching system to identify mutual trade opportunities based on User IDs.
* **Data Export Engine:** Generates downloadable CSV reports for "Missing" and "Duplicates" lists, formatted with Sticker Codes and Team Names.
* **Interactive Analytics:** Plotly-powered dashboard showing completion percentages and team-specific progress metrics.

## 🛠️ Tech Stack
* **Python:** Core inventory logic and backend integration.
* **Streamlit:** UI framework for the interactive web experience.
* **Supabase:** Cloud-based PostgreSQL database for persistent storage and Auth.
* **Pandas:** Data manipulation for exporting reports and processing trades.
* **Plotly:** Dynamic rendering of financial-style growth and distribution charts.
